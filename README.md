# AngularHotel

本系统作为蚌壳民宿平台项目的web资源管理端，使用ng开发的单页应用系统
主要内容是：

 * 数据统计（echarts图视化）
 * 管理用户信息
 * 管理房间信息
 * 管理投诉信息
 * 管理订单信息
 * 管理地区信息
 * 系统管理（权限管理）
    
    
#技术

1.后端
* 核心框架：Spring Framework 4.2.5
* 视图框架：Spring MVC 4.2.5
* 持久层框架：MyBatis 3.2

2.前端
* AngularJS 1.3
* BootStrap 3
